# Birdwalk RN - Development Roadmap

## Overview
This roadmap tracks the planned features and improvements for the Birdwalk React Native application.

## Priority Levels
- **P0**: High priority - Core features that should be implemented soon
- **P1**: Medium priority - Important features for enhanced functionality
- **P2**: Lower priority - Nice-to-have features for future consideration

---

## Tasks

### P0 - High Priority

#### 1. Add Sort to Lifers Tab
**Status**: Planning
**Plan Document**: [plans/01-lifers-sort.md](plans/01-lifers-sort.md)

Implement sorting functionality for the Lifers tab to allow users to organize their lifer bird sightings by different criteria.

#### 2. Dark Mode
**Status**: Planning
**Plan Document**: [DARK_MODE_PLAN.md](DARK_MODE_PLAN.md)

Add dark mode support to provide a dark color scheme option for users, improving usability in low-light conditions.

#### 3. Offline Mode
**Status**: Planning
